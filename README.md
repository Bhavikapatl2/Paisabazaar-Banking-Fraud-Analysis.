This project helped me understand how banking data can be analyzed to identify customer behavior, credit risk, and financial patterns. I learned to work with large datasets using Python, Pandas, and NumPy, and to create meaningful visualizations with Matplotlib and Seaborn. By exploring features like income, credit score, EMI, and payment habits, I gained insights into which customers are reliable and which may pose financial risk. The project also taught me how to present data professionally on GitHub, including embedding graphs and creating a clear README for others to easily understand the analysis.
